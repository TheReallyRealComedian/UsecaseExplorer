Inject Areas

Upload a JSON file containing a list of area objects, each with a "name" key.

Example: [{"name": "Area One"}, {"name": "Area Two"}]

{# CSRF token needed if using Flask-WTF/CSRFProtect later #}

Inject Process Steps

Upload a JSON file containing a list of process step objects.

Required keys per object: "bi_id" (string, must be unique), "name" (string), "area_name" (string, must match an existing Area name).

Optional keys: "raw_content" (string), "summary" (string).

Example: [{"bi_id": "PS-001", "name": "Step One", "area_name": "Area One", "summary": "Summary one"}, ...]

{# CSRF token needed if using Flask-WTF/CSRFProtect later #}

Inject Use Cases

Upload a JSON file containing a list of use case objects.

Required keys per object: "bi_id" (string, must be unique for use cases), "name" (string), "process_step_bi_id" (string, must match the BI_ID of an existing Process Step).

Optional keys: "raw_content", "summary", "inspiration" (all strings).

Example: [{"bi_id": "UC-001", "name": "Use Case One", "process_step_bi_id": "PS-001", "summary": "Summary one"}, ...]

{# CSRF token needed if using Flask-WTF/CSRFProtect later #}
